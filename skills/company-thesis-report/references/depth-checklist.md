# Depth checklist — research-grade (beyond “decision-grade lite”)

Use this before PDF delivery. The prior quality bar stops headings-only reports; this
checklist forces **institutional-memo depth**. If more than two items fail, keep
mining sources — do not ship.

## A. Evidence depth (lookback)

- [ ] **Latest quarter fully mined:** results PR + investor deck + **full concall** (not highlights-only)
- [ ] **Trailing lookback:** last **4–6 quarters** of concalls/PRs for guidance evolution, mix, and repeated risks (cache prior extracts; query, don’t re-read wholes)
- [ ] **Annual depth when needed:** latest AR (or prior) queried for segments, related party, contingent liabilities, auditor emphasis — at least when BS/litigation/segment gaps remain
- [ ] **Primary over aggregator:** every material number traces to screener-filed tables, exchange filing, or company transcript/deck

## B. Decision depth

- [ ] Action + confidence + **time horizon** (e.g. 2–3 quarters / 12–18 months)
- [ ] **Key debate** stated in one sentence (the single disagreement a bull and bear would have)
- [ ] Confirm / kill lists are **numeric or observable** next 1–2 prints (not vibes)
- [ ] **Position use:** how to use the call (core hold / satellite / wait / trim) — not financial advice boilerplate; practical sizing *framing* only
- [ ] **Alternative thesis** written (what the other side believes and why it could be right)

## C. Financial depth

- [ ] 5–10y annual history **with narrative** of regime shifts (when growth/margins broke)
- [ ] Last 6–8 quarters chart/table **with seasonality called out**
- [ ] **Earnings / growth bridge:** what drove YoY and QoQ (volume/mix/price/geo/one-offs/other income)
- [ ] **Working capital pack:** debtor days trend, CFO vs PAT, any DSO commentary from call — mandatory even if “clean”
- [ ] **Capital allocation:** dividends, buybacks, M&A appetite, cash vs reinvestment (from call/AR)
- [ ] Lens must-have metrics present or explicitly gapped with search attempted

## D. Operating / KPI depth

- [ ] **KPI scorecard** over ≥4 quarters where disclosed (annuity %, logos, book-to-bill, GNPA, occupancy, etc. — lens-defined)
- [ ] Mix deep-dive: geo **and** product/vertical/stream when both exist
- [ ] Demand: named wins + whether count vs size is changing
- [ ] Management scorecard: what they guided/signaled last 2–4 quarters vs what happened

## E. Market / relative depth

- [ ] ≥3 peers with real multiples **and** 2–4 sentences on premium/discount deserved
- [ ] Valuation states **implied growth** the market is paying for at current PE/EV (back-of-envelope OK if labeled)
- [ ] Bull/base/bear bands tied to explicit growth/margin assumptions (not orphan price targets)

## F. Intellectual honesty

- [ ] Risks include the **thesis-breaking** ones, not only generic macro
- [ ] Moats split real vs marketing
- [ ] Gaps stated where data missing (no silent omit)
- [ ] Would a skeptical PM say “this is enough to allocate capital”? If no → deepen
